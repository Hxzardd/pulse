# Pulse App

Prototype frontend for patent: "System and Method for Adaptive Social Media Interfaces using Real-Time Interaction Kinematics for Cognitive Fatigue Mitigation"

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## What It Does

Detects doom-scrolling behavior (high scroll velocity + low dwell time) and automatically switches to a low-stimulation "Cozy Mode" with increased scroll friction and grayscale filters.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
